# Cover photos checklist

Drop your chosen photos into this folder (`assets/covers/`) using **exactly** these filenames
so they show up automatically on each article page, on the homepage and on the category pages.

Recommended specs: **JPG or WEBP, ~1200x630px, under ~300KB** (keeps the site fast, which
matters for AdSense approval and Core Web Vitals). Until a file is added, that article will
show a colored placeholder with a reminder instead of a broken image — nothing looks broken.

**License checklist before you download any photo:**
- Must allow **commercial use** (this site runs ads): CC0, CC BY and CC BY-SA are fine — **not CC BY-NC**.
- If the license requires attribution (CC BY / CC BY-SA), add the credit line in the article's
  `cover_credit` field in `site_build/articles_data_*.py` (search for `cover_credit`), e.g.
  `"Photo by Jane Doe on Unsplash"`, then re-run `python3 build_site.py`.
